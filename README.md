# Bottle + websockets.

Ejemplo de uso de comunicaciones **websockets** usando **bottle** (cumple con la espacificación de **WSGI** -Web Server Gateway Interface- de python).

### Dependencias.
La aplicación requiere la instalación de:
 * bottle.
 * gevent-websocket.
